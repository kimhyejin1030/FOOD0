# 🖥️ 프로젝트 소개
수기 리스트 작성 및 오랜 대기 필요없이 고객이 사이트를 통해 가게에 대기를 걸어놓고 매장에서는 사이트를 통해 해당 순번 고객을 호출할수 있는 음식점 대기 예약 사이트로 [지역별 가게 검색],[메뉴추천] 기능도 제공합니다. 

<br>
